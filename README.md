# Catcherry - платформа геймификации рабочих и учебных процессов

## Формулировка решаемой проблемы

В условиях цифровой экономики остро встаёт вопрос внутренней и внешней мотивации студентов и работников организаций различного размера. Падает качество и эффективность дистанционного обучения, работники распределённых команд тяжелее мотивировать. Практика современного менеджмента преполагает использование геймификации и игровых методик.

Крупные корпорации всё чаще и чаще заказывают разработку индивидуальных решений, стоимость которых недостижимо высока для среднего и малого бизнеса. Однако вне размера организаций и количества сотрудников, проблемы остаются схожие, но на их решение уже не остаётся бюджета.

Catcherry - стартап-проект, придуманный и создаваемый студентами Московского политеха, направленный на создание платформы-конструктора, позволяющего выстроить свою геймификационную систему используя различные игровые модели и методики.

Практическое применение игровых методик в неигровых контекстах является акутальным и малоизученным вопросом научного сообщества. Проект подразумевает научно-исследовательские работы, а потому одним из приоритетов проекта является создание не финального продукта, а базы для проведения таковых исследований, с целью установить зависимости между характеристиками пользователей от качества применения тех или иных моделей.

Стартап планирует продвижение не только в России и странах СНГ, но и в различных странах, а потому команде проекта необходимо уделить внимание интернационализации, конвертации дат, валют и т.п.

В дальнейшем планируется внедрение технологий машинного обучения и анализа с целью уникальной аналитики применения игровых методов.

## Цель проекта

### Цель I семестра разработки (лето-осень-зима 2020)

Разработать MVP, пригодный к применению, тестированию совместно с преподавателями Московского политехнического университета, исследованию игровых механик и проверке бизнес-теории

## Установка проекта

```
npm install
npm run build
node ./server/server.js

Проект будет запущен на http://localhost:3000/ (по-умолчанию)
```

### Компиляция и хот-релоад для разработки

```
npm run serve
```

### Компиляция и минификация для продкшена

```
npm run build
```

### Запустить unit-тесты

```
npm run test:unit
```

### Запустить end-to-end тесты

```
npm run test:e2e
```

### Исправление и проверка красоты кода

```
npm run lint
```
